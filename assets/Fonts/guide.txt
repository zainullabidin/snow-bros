            - Gem cost labels
            - Button text
            - Power-up names

4. VT323-Regular.ttf
   WHERE  : Dialogue, Notifications & Debug
   USED FOR:
            - In-game notifications ("Enemy Encased!")
            - Power-up pickup messages
            - Bonus level announcement
            - Star event prompt
            - F1 debug mode labels (hitbox info)
            - Loading/transition messages
   SIZE   : 18-24px (larger size needed for VT323
            as it runs small)
   WHY    : Retro terminal feel adds character
            to quick pop-up messages without
            being too formal.

5. Orbitron-VariableFont_wght.ttf
   WHERE  : Main Menu & Character Select Screen
   USED FOR:
            - "SNOW BROS" main title (large)
            - Main menu button labels
            - Character names on select screen
            - Character stat labels
            - Settings screen headings
            - Leaderboard screen title
   SIZE   : 32-64px for title, 16-20px for buttons
   WHY    : Modern, futuristic, premium feel.
            Makes the main menu look polished
            and professional. Good contrast
            against the arcade gameplay fonts.
 

6. Cinzel-VariableFont_wght.ttf
   WHERE  : Boss Encounter Screens & Special Events
   USED FOR:
            - Boss name reveal ("MOGERA AWAKENS")
            - Boss defeated reward screen
            - Currency reward display
            - Star event title
            - End game victory screen
            - Any dramatic/cinematic moment
   SIZE   : 28-48px (large, dramatic)
   WHY    : Luxury/elegant serif font creates
            a dramatic contrast when a boss
            appears. Feels like an epic moment.
            Very different from the arcade fonts
            used everywhere else.
